# 位运算

- `c_XOR`：棋盘格的本质是异或？按位异或画棋盘格。

## 引入索引值 `a` 多取几位，扩展图案类型

- `c_XOR_2`：按位异或
- `c_AND_2`：按位与
- `c_OR_2`：按位或

使用方式：点鼠标改变 `a` 值看不同图形，其中左键加 1 右键减 1。可以对照控制台输出值与画出的图形。按 `s` 键保存图片。

### 预料之外的分形：类似谢尔宾斯基三角形

`c_AND_2` 就能画，就像例子 `c_AND_2/examples/c_AND_2 a_511_111111111.png`。画出这种分形最快的方式，是点 2 次右键，或者直接修改 `a` 的值为 -1 再运行，不论画布多大。

目前发现了两种得到这种分形的方式：

- `c_AND_2a`：从最细节开始  
- `c_AND_2b`：从最大的空白开始  

使用方式：点几下鼠标，对照控制台输出值与画出的图形，看一看这种图形怎么得到的。按 `s` 键保存图片。

### 比较不同位运算的图形

- `c_bitwise_1`：一次只画一种位运算
- `c_bitwise_2`：放一起比较

使用方式：点鼠标改变 `a` 值看不同图形，其中左键加 1 右键减 1。可以对照控制台输出值与画出的图形。按 `s` 键保存图片。  
对 `c_bitwise_1`，按数字 `1`、`2`、`3` 键看不同图形，分别对应按位与、按位或、按位异或，默认按位异或

## 直接填充颜色，画出新图案

位运算结果直接填充颜色，于是得到各种不同灰度

- `c_bitwise_3`：画灰度图，一次只画一种位运算
- `c_bitwise_4`：画灰度图，放一起比较

使用方式：点鼠标改变 `a` 值看不同图形，其中左键加 1 右键减 1。可以对照控制台输出值与画出的图形。按 `s` 键保存图片。  
对 `c_bitwise_3`，按数字 `1`、`2`、`3` 键看不同图形，分别对应按位与、按位或、按位异或，默认按位异或

- `c_bitwise_3a`：图案动态变化，只画灰度，夹杂不同图案划过留下的痕迹
- `c_bitwise_3b`：图案动态变化，不是灰度的都画成不透明，能看到蓝色，如果有足够耐心，能看到其他颜色

使用方式类似与 `c_bitwise_3` 类似。
